# Test-driven development

### Add a test: 1 point!

### Make the test compile: 2 points!

### Make the test pass: 3 points!

You only get the points if it didn't pass beforehand!

[Engine-building](https://www.ign.com/articles/best-engine-building-board-games): Write lots of tests!
